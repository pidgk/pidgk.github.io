Crear un escaner de puertos simple con python3

Hoy vermos una forma bastante sencilla de crear nuestro propio escaner de puertos. Aqui les dejare un ejemplo de codigo con el funcionamiento del escanner.

PY: Python3 code
  1. import socket
  2. host = "192.168.1.1"
  3. for port in range(1000):
  4.        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
  5.        value = sock.connect_ex( (host, port) )
  6.        if value == 0:
  7.                print("Puerto Abierto", port)
  8.        else:
  9.              pass

1.Lo primero que hacermos es importar la libreria socket, para poder realizar conexiones remotas.
2.Despues definimos la direccion address del host que se le van a escanear los puertos.
3.Iniciamos un bucle for con un range en la variable puerto.
4.Creamos una variable que almanecera nuestro socket con los modulos
de AF_INET para especificar ipv4 y SOCK_STREAM para especificar
el protocolo de red TCP/IP.
5.Creamos una variable donde el socket se va a conectar a la direccion
y el rango de puertos.
6 y 7.Con el condicional if verificamos que si la respuesta es correcta
nuestro programa diga que el puerto esta abierto.
8. Si lo anterios no sucede simplemente se pasa por alto.
